== Description ==
Extends the WooCommerce Stripe gateway card management by allowing customers to save new cards without having to make a purchase.

Customer must have checked out one time using Woo Stripe before being able to save new cards.

REQUIRES WOOCOMMERCE GATEWAY STRIPE (http://www.woothemes.com/products/stripe/).

== Installation ==

1. Ensure you have latest version of [WooCommerce]plugin installed
2. Ensure you have latest version of [WooCommerce Stripe Gateway] plugin (http://www.woothemes.com/products/stripe/) installed
3. Activate 'Woo Add-On For Stripe' through the 'Plugins' menu in WordPress

== Usage ==

Adding a new card

1- Customer must have checked out once with WooCommerce Stripe before being able to save new cards using this plugin.

2. Customer visits the "My Account" page.

3. Customer can add new cards using the "Add A Card" section.
